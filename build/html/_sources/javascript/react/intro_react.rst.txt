.. _bootstrap_vs_react:

bootstrap vs react
=====================

- Bootstrap 主要是一个 :ref:`css` 框架，用于设置Web元素的样式(但也包含了简洁的 ``JQuery`` JS库用于构建简单交互)；React则是一个大而全的构建用户界面的 :ref:`javascript` 库，是基于组件的框架，专注于构建可重用的UI元素
- Bootstrap 为单页和动态Web应用提供了现成的模版；React则可以创建动态交互式UI，尤其适合复杂的应用程序
- Bootstrap适合快速构建Web页面样式，创建基本的布局和原型设计；React适合构建复杂、交互和动态的Web应用，特别是单页面(SPA)应用
