Linux服务器
=================================

Linux系统管理的通用性技术：实际上Linux各个发行版主要差异在于软件包管理方式，绝大多数技术都是通用的。由于一些底层基础技术在某个发行版比较倚重，所以我会在具体的发行版中解析这项技术。例如， :ref:`systemd` 就是因为Red Hat最早并且主推systemd作为系统服务的管理器，而且资料完备，所以我单独在 :ref:`redhat_linux` 中汇总。
