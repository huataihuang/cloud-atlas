=================================
ocserv VPN
=================================

在Internet上，安全加密通讯是非常重要基础设置，无论是企业还是个人，都需要使用VPN来保护自由的通讯。

`IPsec VPN Server Auto Setup Scripts <https://github.com/hwdsl2/setup-ipsec-vpn>`_ 是一个快速构建IPsec VPN服务器的脚本，并且提供了相关技术索引，将作为本章节学习实践的起步。
