通过Xpra使用Jetson Nano图形桌面

我最初尝试在Jetson Nano上安装Xpra是通过 Jetson Nano平台编译安装Xpra 实现的，步骤曲折。不过，我在构建 边缘云计算架构(旧版) 时，是通过发行版提供的Xpra完成，所以本文记录方法更为通用。

我在 边缘计算Jetson Nano 做了系统精简之后再部署Xpra，如果你喜欢功能大而全的系统，可以忽略我这个精简步骤。

安装Xpra
