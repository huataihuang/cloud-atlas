在Ubuntu安装NVIDIA CUDA

实践记录:

  • 安装NVIDIA Linux驱动 物理主机安装CUDA driver

  • 在OVMF虚拟机中安装NVIDIA Linux驱动 虚拟机内部安装CUDA driver

  • 树莓派安装NVIDIA P4 GPU运行 nvidia-docker 容器 树莓派ARM架构Ubuntu环境安装CUDA driver成功，但是实践也发现 树莓派 Raspberry Pi OS 安装NVIDIA驱动(归档) (Raspberry Pi OS虽然是debian变种，但是实际上无法正常编译安装CUDA驱动) 失败

  • Bhyve环境Ubuntu虚拟机运行Tesla P4 GPU的Docker 在 FreeBSD 的 bhyve(BSD hypervisor) 虚拟化环境中运行Ubuntu，来构建一个运行 ollama
