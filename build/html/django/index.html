Django

Django是Python编写的开源Web应用框架，结构清晰，应用广泛。在云计算领域中，著名的开源云计算平台 OpenStack 和 oVirt 都采用了Django实现。

学习Django开发也能够帮助我们学习OpenStak云计算，深入理解开源项目的精髓。

备注

其他常用的python web框架可以参考 Python’s Frameworks Comparison: Django, Pyramid, Flask, Sanic, Tornado, BottlePy and More ，其中比较常用的框架还有:

  • Flask : 微型web框架，流行度可能仅次于Django，文档全面 请参考 Flask vs Django: A Detailed Comparison of Python Web Frameworks

  • CherryPy ( Ceph 的 Ceph Manager Daemon 模块 Ceph Dashboard 管控面板 就使用CheeryPy开发) : 轻量级的面向对象极简框架，性能极佳，并且易于学习使用 请参考 CherryPy Vs Flask – Which Is The Better Framework for Python?
