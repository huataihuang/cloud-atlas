================
rclone简介
================

Rclone是一个从不同云计算存储服务同步文件和湖目录的命令行工具，它就像是一个 :ref:`rsync` for cloud storage 程序。

我计划尝试一些常见开源云存储和常用云服务存储:

- :ref:`ceph`
- Nextcloud
- Amazon S3
- Dropbox
- Google Cloud Storage
- Google Drive
- iCloud Drive
- Microsoft Azure Files Storage
- Microsoft OneDrive

由于该工具支持的云平台非常多，我准备做一些实践来丰富我的文件存活和同步方案
