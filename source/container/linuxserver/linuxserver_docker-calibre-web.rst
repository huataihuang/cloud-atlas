===============================
linuxserver/colibre-web
===============================

``linuxserver/colibre-web`` 提供了易于浏览、阅读和狭隘现有 ``Calibre`` 数据库提供的电子书。我计划在 :ref:`linuxserver_docker-calibre` 部署之后结合使用，以实现一个较为完善的电子书网站。

部署计划采用 :ref:`kubernetes` ，所以前期基础工作会比较繁重，进度会比较慢。待续...
