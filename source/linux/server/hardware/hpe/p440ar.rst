======================
HPE P440ar 存储阵列卡
======================

故障排查
=========

我在淘宝购买的二手P440ar非常便宜，只需要25RMB，原本以为能够轻易解决SAS磁盘使用的问题，但是没有想到加电仅半小时，服务器突然宕机。通过 :ref:`hp_ilo` 检查 ``Integrated Management Log`` 有如下日志:

.. literalinclude:: p440ar/roc_1_error
   :caption: p440ar阵列卡无响应出发Runtime Fault宕机

- 启动服务器按 ``F10`` 进入 ``HP Itelligent Provisioning`` / ``HP Smart Storage Administrator`` / ``Scripting Toolkit Windows PE 64 Bit Mode`` 三个选项，选择 ``HP Smart Storage Administrator`` 来管理服务器上存储:

- 过一会能够看到 ``Smart Storage adminstrator`` 界面，

- 我发现开机大约20分钟以后服务器宕机，同样的报错，但是我也注意到在这个报错前有一行服务器刚启动时的日志显示阵列卡没有配置:

.. literalinclude:: p440ar/not_config_log
   :caption: 启动系统日志显示阵列卡没有配置
