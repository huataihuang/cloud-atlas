===========================
个人服务器的思考
===========================

我从2021年10月购买了第一台服务器 :ref:`hpe_dl360_gen9` 开始，慢慢购置了不少服务器硬件和组装台式机、 :ref:`raspberry_pi` ，中间折腾不断，也踩了不少坑，逐渐也有了一些"垃圾佬"的心得体会。

建议一: 二手品牌服务器为主
============================

我在折腾 :ref:`nasse_c246` 组装机之后，才发现真正性价比高的是海鲜市场淘汰的二手服务器: 全尺寸主板和高质量的组合机箱，使得安装扩展设备极为方便，并且因为市场保有量大，价格和质量实际上比杂牌新电脑配件强很多。

以 :ref:`dell_t5820` 为例，我没有想到二手准系统只有区区1100元，就能获得:

- 金牌950W电源
- 设计巧妙，散热极佳的静音机箱，秒杀一众昂贵的组装机箱和散热风扇
- 绝对高质量的品牌主板，服务器类主板即使二手稳定性也比家用的新主板更好

如果个人从淘宝筛选组合上述同等质量的准系统，价格预估要翻翻

.. note::

   没有海量的流水线生产，现在装机市场的攒机已经无法达到品牌主机的性价比了

建议二: 省钱用机架服务器，但静音工作站最佳
===========================================

二手市场上，最经济实惠的是机房下架的机架式服务器: 数量庞大导致高性能机架服务器只有几百到几千就能获得原先上万甚至十几万才能获得的服务器，性价比无敌。

然而，机架服务器最大的问题是噪音，不论怎样调试你都很难将机架服务器24小时在家里运行，噪音和恐怖的耗电会让你心力交瘁。

我一直想要有一个类似 :ref:`mac_mini_2024` 这样能够24小时运行且接近无声的服务器，我最终的决定是购买 :ref:`dell_t5820` 这样价格低廉性能虽然受限，但是依然性价比极高的工作站。
