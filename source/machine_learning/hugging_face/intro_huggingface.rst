==========================
Hugging Face简介
==========================

`Hugging Face <https://huggingface.co/>`_ 作为AI界的GitHub，是目前全球最流行的开源 AI 社区和平台。它不仅是一个存储模型的地方，更是一套完整的 AI 开发生态系统:

- Models（模型库）：存放各种预训练模型（如 Llama 3.3、Qwen 2.5、Stable Diffusion）。你可以找到原始权重，也可以找到各种量化版本（GGUF, EXL2, AWQ）。
- Datasets（数据集）：存放用于训练或微调 AI 的文本、图像、音频数据。
- Spaces（展示空间）：类似于一个小型的在线 Demo，允许开发者用简单的代码（如 Gradio 或 Streamlit）部署并展示他们的 AI 应用。
- Documentation（文档）：这里有极其详尽的教程，特别是关于 transformers 库的使用。
