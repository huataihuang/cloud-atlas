====================
AMD ROCm简介
====================

AMD ROCm 是一个在AMD Instinct 加速卡和AMD Radeon GPU上运行HPC和AI工作负载实现兼容工业软件框架。

ROCm是一个软件堆栈，主要由开源软件构成，从底层内核到高层最终用户应用程序，提供了对AMD GPU的编程工具:
