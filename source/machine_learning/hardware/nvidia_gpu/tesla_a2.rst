=============================
NVIDIA Tesla A2 GPU运算卡
=============================

NVIDIA A2 Tensor Core GPU是NVIDIA 2021年11月10日推出的低功耗边缘计算推理设备。A2的特点是:

- 低功耗: 功耗只有40-60W(TDP)
- 小尺寸: 半高无风扇(安装在机架服务器)，支持大规模部署
