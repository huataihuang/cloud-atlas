=================================
Ollamas集成Claude Code
=================================

Claude Code 是 Anthropic 的代理编程工具，可以阅读、修改和执行在工作目录下的代码。通过Ollama的Anthropic兼容API能够使用 ``glm-4.7`` , ``qwen3-coder`` , ``gpt-oss`` 模型来用于Claude Code。
