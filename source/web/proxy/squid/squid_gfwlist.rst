.. _squid_gfwlist:

========================
Squid 结合 ``gfwlist``
========================

:ref:`squid_socks_peer` 通过手工来维护squid转发翻墙列表是一个非常麻烦而且无趣的事，好在社区有人维护了一个 `gfwlist (GitHub) <https://github.com/gfwlist/gfwlist>`_ 。通过将这个列表引入Squid，可以定期更新 :ref:`squid_socks_peer` 来维护一个合适的转发配置。

.. note::

   先挖坑，后续更新
