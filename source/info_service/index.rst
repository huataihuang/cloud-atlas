Info-Service
=================================

虽然我的工作聚焦在基础设施，但是，最终的应用运行是 :ref:`infra_service` 以及 IaaS ( :ref:`openstack` 和 :ref:`kubernetes` ) 能力的体现。

规划构建应用服务:

- 视频服务
- 邮件服务
- 即时消息服务
- 协作服务
